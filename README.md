simplessl
=========

a simple ssl solution . Including c/c++, java, python, object-c.

本项目是对ssl的基本封装, 方便开发者使用.包括多种语言的加解密及ssl等,如c/c++, java, python, object-c.


项目目前还处于开发前期,还有很多工作要做.但是作者目前没有太多时间用于此项目开发.
